## ▶️ Running a Project

1. Navigate into the project folder
   ```bash
   cd assignment1
   ```

2. Install dependencies
   ```bash
   flutter pub get
   ```

3. Run on a connected device or emulator
   ```bash
   flutter run
   ```

> Some projects using Realm may require running the code generator first:
> ```bash
> dart run realm generate
> ```
